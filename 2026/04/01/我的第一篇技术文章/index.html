前言

这是我的第一篇技术博客！今天我想分享一下学习 JavaScript 的一些心得。

为什么学习 JavaScript？

JavaScript 是 Web 开发的三大核心技术之一（HTML + CSS + JavaScript），几乎每个网站都在用它。

主要用途：

  • 前端交互：点击按钮、表单验证、动画效果
  • 后端开发：Node.js 让 JS 可以写服务器
  • 移动应用：React Native、Ionic
  • 桌面应用：Electron（VS Code 就是用它写的）

学习建议

  1. 多动手：看 10 遍不如写 1 遍

  3. 查文档：MDN 是最好的老师
