🎉 鼠标跟随爱心粒子效果

一个浪漫的鼠标跟随爱心粒子效果，为你的博客增添温馨的交互体验！

✨ 功能特性

🎯 核心功能

  • 鼠标跟随：爱心粒子跟随鼠标轨迹生成
  • 点击爆发：点击时在鼠标位置爆发多个粒子
  • 触摸支持：完美支持移动端触摸操作
  • 自动清理：粒子自动消失，不会影响性能

🎨 视觉效果

  • 多种颜色：粉色、红色、紫色、蓝色等多种爱心颜色
  • 不同大小：小、中、大三种尺寸的爱心
  • 随机动画：3种不同的飘浮动画效果
  • 发光效果：爱心带有柔和的发光和阴影效果

⚙️ 智能控制

  • 开关控制：右侧爱心按钮可开启/关闭效果
  • 设置记忆：用户偏好自动保存到本地存储
  • 性能优化：移动端自动减少粒子数量
  • 节流处理：鼠标移动事件节流，避免性能问题

🌙 主题适配

  • 深色模式：自动适配深色主题
  • 响应式设计：完美适配各种屏幕尺寸
  • 无障碍支持：支持减少动画偏好设置
  • 高对比度：支持高对比度模式

🚀 使用方法

1. 文件已集成

所有必要的文件已经集成到你的博客中：

  • CSS样式已添加到 source/_data/styles.styl
  • JavaScript已添加到 source/js/heart-particles.js
  • 布局文件已更新，自动加载脚本

2. 自动启动

效果会在页面加载完成后自动启动，无需额外配置。

3. 用户控制

  • 用户可以通过右侧的爱心按钮开启/关闭效果
  • 设置会自动保存，下次访问时记住用户偏好

🎮 交互说明

鼠标操作

  • 移动鼠标：生成跟随轨迹的爱心粒子
  • 点击屏幕：在点击位置爆发3-8个爱心粒子
  • 悬停按钮：爱心控制按钮有悬停动画效果

触摸操作（移动端）

  • 触摸移动：生成跟随触摸轨迹的爱心粒子
  • 点击屏幕：在触摸位置生成爱心粒子

控制按钮

  • 位置：固定在屏幕右侧中央
  • 状态：实心爱心（开启）/ 空心爱心（关闭）
  • 提示：悬停时显示当前状态提示

⚡ 性能优化

自动优化

  • 粒子限制：最大50个粒子（移动端25个）
  • 节流处理：鼠标移动事件50ms节流（移动端100ms）
  • 自动清理：粒子3-3.5秒后自动消失
  • 内存管理：定期清理过期粒子引用

用户偏好

  • 减少动画：支持 prefers-reduced-motion 设置
  • 页面隐藏：页面不可见时自动暂停
  • 性能监控：移动设备自动降低粒子数量

🐛 故障排除

效果不显示

  1. 检查浏览器控制台是否有JavaScript错误
  2. 确认 heart-particles.js 文件已正确加载
  3. 检查是否启用了 prefers-reduced-motion

性能问题

  1. 减少 maxParticles 数量
  2. 增加 throttleDelay 延迟
  3. 在移动端禁用效果

样式问题

  1. 检查CSS文件是否正确加载
  2. 确认没有其他样式冲突
  3. 检查深色模式适配

🌟 特色亮点

  • 零配置：开箱即用，无需复杂设置
  • 用户友好：提供直观的开关控制
  • 性能优秀：智能优化，不影响页面性能
  • 视觉精美：多种颜色和动画效果
  • 兼容性强：支持所有现代浏览器
  • 响应式：完美适配桌面端和移动端

💡 使用建议

  1. 适度使用：建议在特殊页面（如首页、关于页）使用
  2. 用户选择：保留用户关闭效果的选项
  3. 性能监控：在低性能设备上考虑默认关闭
  4. 主题协调：确保爱心颜色与网站主题协调
